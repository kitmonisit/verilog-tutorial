## Getting icarus-verilog

    cd dev
    git clone https://github.com/steveicarus/iverilog.git icarus-verilog

## Compiling and installing

    cd icarus-verilog
    git checkout v0_9_5
    sh configure
    make check
    make install

## Uninstalling

    cd dev/icarus-verilog
    make uninstall
